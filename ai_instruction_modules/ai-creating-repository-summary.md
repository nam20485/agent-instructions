# Creating Repository Summary

## Goal
Create `ai-repository-summary.md` at repository root to onboard Copilot agents efficiently.

## Purpose  
Reduce agent exploration time, minimize build/validation failures, prevent PR rejections.

## Requirements
- **Max length**: 2 pages
- **Scope**: General (not task-specific)
- **Content focus**: High-level details + build instructions

## What to Include

### High-Level Details
- Repository purpose/description
- Project type, size, languages, frameworks, runtimes
- Architecture overview

### Build Instructions  
- Bootstrap, build, test, run, lint commands
- Tool versions and prerequisites
- Common errors and mitigations
- Environment setup steps

### Validation
Test all documented commands in clean environment to ensure accuracy.
